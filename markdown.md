# Italic

\*italic text\*\
or\
\_italic text\_

Example: *italic*


# Bold

\*\*bold text\*\*\
or\
\_\_italic text\_\_

Example: **bold**


# NewLine

To add new line, either
* end a line with two or more spaces
* end the line with a '\\'

From the markdown specification:
>When you do want to insert a `<br />` break tag using Markdown, you end a line with two or more spaces, then type return.
